China Crude Imports and Crude Storage¶

This notebook will give a tutorial on how to query our data through pythonSDK.

In this notebook, we will query data for Time series of China oil imports and crude storage (floating storage and onshore crude storage) as our example.

The steps required are following:

  1. Import Libraries
  2. Extract product ids
  3. Query raw cargo movements data (table view)
  4. Query cargo movements data (time series view)
  5. Floating storage, Crude imports, Onshore crude inventories

    filter_time_max=datetime.today(),

    filter_destinations = china,
    #filter_origins = us,
